# Sunset Apartment Gardony

Weboldal a Sunset Apartment Gardony szallashelynek -- a Velencei-to partjan.

**Tech stack:** Next.js 16, React 19, TypeScript, Tailwind CSS 4

## Oldalak

- `/` -- Fooldal
- `/apartmanunk` -- Apartmanunk (galeria, leiras, felszereltseg, terkep)
- `/ajanlataink` -- Ajanlataink es arak
- `/szobak` -- Szobak
- `/kapcsolat` -- Kapcsolatfelvetel
- `/adatvedelem` -- Adatvedelmi tajekoztato

## Fejlesztes

\\\ash
npm install
npm run dev
\\\`n
Open http://localhost:3000

## Build

\\\ash
npm run build
npm run start
\\\`n
## Deploy

Vercel-re egyszeru deploy: importald a repot a Vercel Dashboardon (https://vercel.com/new).
